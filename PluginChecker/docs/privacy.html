Privacy

The application makes online requests to an API (Plugin Checker Index / PCI) which is also authored by me.

The service provider on which the API relies (Vercel), may collect some data.
However, the API in itself does not collect any data.
You can check Vercel's privacy policy here.

Furthermore, the Plugin Checker application in itself does not collect any data at all.
It only stores the obtained data from the API locally on your system.
This data is purely functional and solely related to GTA V / LSPDFR plugins. It gets stored until newer data is available, which will then replace the existing data.

It does not involve the end user's personal data whatsoever.
You can check for yourself in the %AppData%/Roaming/PluginChecker/cache directory.
