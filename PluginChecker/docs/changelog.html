Changelog v0.2.0b

This page contains a list of all changes which have been made in this version.
  • Added: Version incompatibilty analysis module which now allows you to automatically detect outdated versions of LSPDFR plugins and GTA V dependencies!

  • Improved: The caching system, data now stays cached until a new verson of it is available.
  • Improved: The overall UI by making minor changes to it and enlarging the font.
  • Improved: Application performance and cleanliness within the source code by making minor changes to the backend.
  • Removed: Community support columns within the grids, as this data was rather subjective.
